# AskMkiM_CrossPlatform

**AskMkiM_CrossPlatform** — кроссплатформенная настольная экосистема нового поколения для работы с системами АСК-МКИ.  
Проект включает современный интерфейс на Avalonia, модульное ядро логики, уровень данных на SQLite и инструменты для полной конфигурации, администрирования и протоколирования оборудования.

Цель проекта — создать единое приложение, работающее на Windows, macOS и Linux, обеспечивающее удобное управление всеми компонентами аппаратного комплекса.

---

## 🚀 Основные возможности

### 🔹 Кроссплатформенный интерфейс на Avalonia
- Единый UI на всех операционных системах  
- Высокая производительность  
- Поддержка светлой/тёмной темы  

### 🔹 Модульное ядро логики
- Чистая архитектура  
- Разделение на подмодули Core, Data и UI  
- Расширяемость и тестируемость  

### 🔹 Локальная база данных SQLite
- Хранение логов  
- Пользовательские настройки  
- Конфигурации оборудования  
- EF Core 8, миграции, репозитории  

### 🔹 Административная панель АСК-МКИ
- Редактор настроек оборудования  
- Параметры приложения  
- Система протоколирования  
- Настройки метрологии  
- Инструменты диагностики

```

│
├── src/

```

## 📦 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/GameShowLDA/AskMkiM_CrossPlatform.git
cd AskMkiM_CrossPlatform
```

### 2. Восстановление зависимостей
```bash
dotnet restore
```

### 3. Запуск приложения
```bash
dotnet run --project src/AskMkiM.Desktop
```
